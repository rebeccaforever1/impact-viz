# Program Impact Dashboard

A data-rich program evaluation dashboard intended for public agencies. This dashboard demonstrates LBS Ventures' capabilities in transforming program data into actionable insights through modern data visualization and analytics.

## Live Demo

**View the dashboard:** [https://rebeccaforever1.github.io/impact-viz/](https://rebeccaforever1.github.io/impact-viz/)

## Features

- **Multi-Tab Analytics**: Comprehensive views across program overview, participant demographics, and impact outcomes
- **Real-time KPI Monitoring**: 18+ key metrics with trend sparklines and progress indicators
- **Interactive Visualizations**: 8+ charts per tab with click-to-expand functionality
- **Equity-Focused Analysis**: Demographic distribution and access metrics
- **Sortable Data Tables**: Detailed program performance with multi-column sorting
- **Professional Design**: Government-appropriate styling with sophisticated color palette
- **Responsive Layout**: Optimized for desktop and tablet viewing

## Tech Stack

- **Frontend**: React 18, TypeScript
- **Charts**: Recharts
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Create React App

## Dashboard Sections

### Program Overview
- Performance trends and efficiency metrics
- Program comparison and completion rates
- Satisfaction tracking and capacity utilization

### Participant Analysis  
- Geographic distribution and growth metrics
- Access & inclusion scores across demographics
- Participation trends by region

### Impact & Outcomes
- Employment outcomes and earnings progression
- Economic impact analysis and ROI metrics
- Program-specific outcome comparisons
